What is Git?

Git is a version control system that helps developers track and manage changes to their code. It allows users to record updates, go back to previous versions, and work on different features without losing progress. Git is especially useful when many people are working on the same project because it keeps everything organized and prevents conflicts.


In summary:

  • Git is a free and open source distributed version control system designed to handle projects with speed and efficiency.
  • Git maintains a local copy of the entire repository with history and version-tracking abilities.
  • Git is easy to learn.

What is GitHub?

GitHub is an online platform that hosts Git repositories in the cloud. It makes it easier for developers to share their projects, collaborate with others, and store their code safely. GitHub also allows users to contribute to open-source projects and review each others work, making teamwork and learning more efficient.


In summary:

  • GitHub is used to store, manage, and collaborate on code, documents, and other project files.
  • Allows multiple developers to work on a project simultaneously.
  • Provides tools like issue trackers and task boards to help manage projects and assingn work.
  • It serves as a central cloud-based location to store a project's code.

Creating SSH Keys

  • SSH Keys are used to provide a secure and convinient method for connecting to GitHub.
  • This authentication method uses:
    • Private Key- remains on your local machine.
    • Public Key- which you can add to your GitHub account.
# STEPS COMMAND
1 In your terminal, generate a new key pair if needed ssh-keygen -t ed25519 -C "your_email@example.com"
2 Look for where the identification is saved then copy.
3 type cat [paste identification].pub
4 Go to GitHub settings -> SSH and GPG keys -> New SSH key
5 Paste generated SSH key.
