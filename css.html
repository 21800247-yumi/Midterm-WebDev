What is CSS?

  • Cascading Style Sheets
  • CSS is used in web development to control the presentation and formatting commonly in HTMl.
  • External CSS- styles defined in a separate .css file linked to the HTML document.
  • Internal CSS- embedded within the <head> section using a <style> tag.
  • Inline CSS- styles are spplied directly to an idivisual HTML element using style attribute.

CSS Properties

Text and Font

Color Sets the text color.
font-family Specifies the font to be used.
font-size Controls the size of the text.
font-weight Defines the thickness of the text.
text-align Sets the horizontal alignment of text.
text-decoration Adds decoration like underlines or strikethroughs.

Background

background-color Sets the background color of an element.
background-image Specifies a background image.
background-repeat Controls how a background image repeats.

Box Model Properties

width, height Sets the dimentions of an element.
margin Defines the space outside an element\s border.
padding Defines the space between an element's content and border.
border Sets the style, width, and color of an element's border.

Positioning and Layout Properties

display Controls how an element is rendered. (block, inline, flex, grid)
position Specifies the positioning method for an element.(static, relative, absolute, fixed)
float Aligns an element to the left or right, allowing content to wrap around it.
Flexbox layout flex-direction, justify-content, align-items
