What is HTML?

  • HyperText Markup Language
  • Standard language used to create and design the structure of web pages.
  • HTML tells the browser how to display text, images, videos, and other medias.
  • HTML consists of a series of elements.

Basic HTML Tags

Document Structre

Tags Description
<!DOCTYPE html> Defines the HTML version.
<html> Root element of the page.
<head> Contains the meta information.
<title> Sets the title in the browser tab.
<body> The main content of the website.

Text and Content

Tags Description
<h1 to h6> Headings
<p> Paragraph
<br> Line break
<hr> Horizontal line
<strong> Bold text
<em> Italic text
<a href=""> Link
<img src="" alt=""> Image
<sub> Subscript text
<sup> Superscript text
<q> Quotation

Example Outputs:

This is a heading.

This is a paragraph.

This is a bold text.
This is an italic text.
This is a link.
This is a subscript text. This is a superscript text.
These are quotations.

Lists

Tags Description
<ul> Bullet form list.
<ol> Numbered Form.
<li> List item (used inside ul or ol).
