What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It defines how text, images, links, and other elements are arranged and displayed in a browser. HTML uses a series of tags to organize content, making it easy for both developers and browsers to understand the page layout.


In summary:

  • HyperText Markup Language
  • Standard language used to create and design the structure of web pages.
  • HTML tells the browser how to display text, images, videos, and other medias.
  • HTML consists of a series of elements.

Basic HTML Tags

HTML tags are the building blocks of a web page. They define the structure and meaning of content, telling the browser how to display text, images, links, and other elements. Each tag usually comes in a pair — an opening tag and a closing tag — that wraps around content to give it a specific function or style. Here is a lists of basic and commonly used HTML tags.

Document Structre

Document structure is important because it helps web browsers and developers understand how the content of a web page is organized. A proper structure ensures that each part of the page — such as the head, title, and body — has a clear purpose and function. It also makes websites easier to read, maintain, and display correctly on different devices. In short, a well-structured HTML document creates a strong foundation for any website.

Tags Description
<!DOCTYPE html> Defines the HTML version.
<html> Root element of the page.
<head> Contains the meta information.
<title> Sets the title in the browser tab.
<body> The main content of the website.

Text and Content

Text and content are essential parts of a webpage that provide information to the users. They help deliver the message, explain ideas, and make the page meaningful and engaging.

Tags Description
<h1 to h6> Headings
<p> Paragraph
<br> Line break
<hr> Horizontal line
<strong> Bold text
<em> Italic text
<a href=""> Link
<img src="" alt=""> Image
<sub> Subscript text
<sup> Superscript text
<q> Quotation

Lists

Lists are used in HTML to organize items in a structured way, either in numbered (ordered) or bulleted (unordered) format.

Tags Description
<ul> Bullet form list.
<ol> Numbered Form.
<li> List item (used inside ul or ol).

This is a paragraph.
